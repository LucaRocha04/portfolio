This repository contains my personal portfolio website, built with HTML and Bootstrap, showcasing my software development projects.

## Features
- Dark corporate design
- Responsive layout
- Direct links to GitHub repositories
- Sections for About Me, Projects, Tech Stack, and Contact

## Tech Stack
- HTML
- CSS
- Bootstrap
- GitHub Pages
